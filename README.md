# Gixie Clock WebApp

A web version of the [Gixie Clock](https://azurest.shop/products/gixieclock) （淘宝：[钛电优创](https://shop125139370.taobao.com/) - [辉光钟](https://item.taobao.com/item.htm?id=581354199123)）app allows you to control the clock from your computer. Created by reverse engineering its [APK](https://play.google.com/store/apps/details?id=uni.UNICB90ED7).

## Usage
Download the project and open index.html directly.
